# IA

- Antes de machine (determinismo)
    Algoritmos predefinidos, ya sabemos la salida a partir de cierta entrada
    esta interpretabilidad o capacidad de predecir el output va disminuyendo mientras
    el modelo se va a haciendo complejo.
    Rigidez.


- [Machine Learning](math/proba/Machine%20Learning.md): Primer capa de abstracción
    Un proceso en el que a partir de una entrada $ x $ obtenemos una salida $ y $
    Algoritmos que se ajustan mayormente a los datos
        - [Regresion lineal](regresion-lineal.md)
        - [Clasificación](clasificacin.md)
            Agrupacion de clases
                [KNN](knn.md): K Nearest Neighbors

    

- Deep learning
    Simplemente se encuentran patrones, no sabemos como se hace la inferencia para obtener esa salida
    Capa extra de abstracción

    

Automatización de la automatización, ¿cómo aprenden las máquinas?
