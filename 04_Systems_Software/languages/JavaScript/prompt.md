El prompt es una forma de pedirle datos al user directamente desde [[JavaScript]], sin el uso de tecnologias como HTML, lo pide mediante una alerta, y se puede seleccionar el tipo de dao que quieres que lea mediante los objetos: 
1. `readLine(prompt)`: Reads a string value from a user  
2. `readInt(prompt)`: Reads an integer value from a user 
3.  `readFloat(prompt)`: Reads a float value from a user
